# GUI for NISTRS
This program is a graphical wrapper for [NistRs](https://github.com/Omnissi/NISTRS).

## Quick start
```bash
git clone https://github.com/Omnissi/GuiNistRs.git
cd GuiNistRs
cargo run -r
```
